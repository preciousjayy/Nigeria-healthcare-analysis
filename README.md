# Nigeria Healthcare Analysis

A data analysis project exploring healthcare access, facility distribution, and health indicators across Nigerian states. This project was built as part of a personal data analytics portfolio, combining Python, Excel, and data visualisation techniques to uncover insights into healthcare gaps in Nigeria.

---

## Project Overview

Nigeria faces significant challenges in healthcare access, with wide disparities between states in terms of facility availability, staffing, and health outcomes. This project investigates those disparities using publicly available data, with the goal of surfacing actionable insights that could inform healthcare planning and policy.

**Key questions explored:**
- How are health facilities distributed across Nigerian states?
- Which states have the lowest access to healthcare infrastructure?
- What patterns exist between facility availability and population size?
- Where are the most critical gaps in healthcare coverage?

---

## Dataset

- **Source:** Nigeria Health Facility Registry / WHO Nigeria Health Indicators
- **Coverage:** All 36 states and FCT Abuja
- **Key variables:** Facility type, ownership, state, geopolitical zone, population estimates
